# MMTTY_XMMT_OCX_VB6_Csharp_Interop
MMTTY XMMT.OCX VB6 -> C# Interop demonstrates Spectrum control updates from MMTTY and method to connect old VB6 ActiveX to modern .NET GUI environments that do not support old VB6 ocx (unsafe) data types.

Project developed on VS2026 IDE with the help of Gemini & ChatGPT AI

See /XMMTwrapper1/XMMT_OCX_VB6_C#_Interop.md for outline of issues and solutions found.
